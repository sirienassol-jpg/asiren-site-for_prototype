# Дані для прототипу web-застосунку Asiren

## Опис

У цій папці збережено дані для намальованих прототипів web-застосунку **Asiren**.

Додано два варіанти таблиці:

1. **Варіант 1 — дані для сторінок прототипу**  
   Таблиця побудована за принципом: у першому стовпчику вказані назви даних, а у наступних стовпчиках — дані для сторінок сайту.

2. **Варіант 2 — товари меню**  
   Таблиця створена за прикладом викладача: категорія, назва товару, вартість, артикул, статус наявності, опис і кнопка.

## Посилання на Google Sheets

Вставити посилання на таблицю Google Sheets:

**Google Sheets:** ВСТАВИТИ_ПОСИЛАННЯ_НА_ТАБЛИЦЮ

## Файли

- `asiren_two_table_variants.xlsx` — таблиця з двома варіантами даних;

- `README.md` — опис виконаної роботи.
